# ALKY WALLET

ALKY WALLET es una aplicación web de simulación de billetera digital diseñada para gestionar transacciones bancarias sencillas en un entorno seguro y fácil de usar.

## Descripción

Este proyecto permite a los usuarios iniciar sesión, consultar su saldo, realizar depósitos, enviar dinero y revisar el historial de transacciones. Está pensado para presentar una interfaz clara y funcional para operaciones financieras básicas.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Principales funcionalidades

- Inicio de sesión de usuario
- Visualización del saldo disponible
- Depósito de dinero
- Envío de dinero a otros usuarios
- Registro y consulta de transacciones realizadas
